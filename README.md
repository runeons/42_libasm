# 42 Libasm

#### Usage
    make && ./libasm_test
#### Overview
    This is a 42 project whose aim is to get familiar with assembly language.
#### Introduced concepts
⭐  Memory registers  
⭐  ASM instructions
#### Description
- Write an ASM 64 bits library that contains the following functions :
    - ft_strlen
    - ft_strcpy
    - ft_strcmp
    - ft_write
    - ft_read
    - ft_strdup
- Write a main.c to test the above functions.
#### Restrictions
- ASM 64 bits
- Intel syntax (vs. AT&T)
- compile with nasm
- No inline ASM
